Catálogo de Jogos Pessoais:

O projeto é uma aplicação web front-end (construída com HTML, CSS e JavaScript) que funciona como um portfólio ou catálogo interativo focado nos jogos que você já jogou. O conteúdo da página é gerado dinamicamente através de um banco de dados local em formato JSON.O projeto fez parte da imersão de front-end da Alura

1. Estrutura de Dados (data.json) O arquivo JSON armazena as informações do seu catálogo, contendo 7 jogos. Cada jogo possui os seguintes atributos registrados: nome, uma descrição curta, ano de lançamento, um link (para vídeos ou gameplay no YouTube), uma segunda descrição mais detalhada focada na história, e a URL de uma imagem de capa. Os jogos inclusos são:



  • Sekiro

  • GTA V

  • Assassin's Creed Brotherhood

  • Red Dead Redemption 2

  • Metal Gear Solid V: The Phantom Pain

  • Watch Dogs 2

  • Resident Evil 7 biohazard


    • 2. Funcionalidades Principais (script.js e index.html)

    Exibição Dinâmica: O JavaScript faz a leitura do data.json e cria automaticamente os "cards" (artigos) para cada jogo na tela principal.

    Busca em Tempo Real: Há uma barra de pesquisa no cabeçalho. Conforme você digita o nome de um jogo, a tela é atualizada instantaneamente para mostrar apenas os jogos que correspondem ao texto.

    Modal Interativo (Pop-up): Ao clicar em qualquer card de jogo, a tela escurece e um "Modal" aparece sobrepondo o conteúdo. Este modal exibe a história completa do jogo (descricao2), a imagem e um botão com link para assistir à gameplay.

    Sidebar (Barra Lateral): Uma seção fixa à direita que destaca os "melhores jogos de 2025 que eu joguei", contendo uma recomendação especial e detalhada para o jogo Red Dead Redemption 2.


    3. Design e Interface (style.css)

    Identidade Visual: O site possui um tema escuro (Dark Mode) com fundo preto e utiliza tons de roxo como cor primária e de destaque. A fonte utilizada para os textos é a "Quicksand".

    Animação de Fundo: O projeto conta com um efeito visual no fundo da página, criando uma animação em cascata (pingos caindo) feita puramente com CSS (@keyframes fall) e dividida em várias colunas com durações diferentes.

    Responsividade: O layout foi projetado para se adaptar a diferentes tamanhos de tela. Em celulares (telas menores que 480px), a estrutura muda para organizar o conteúdo em uma única coluna e oculta o rodapé.

    Rodapé: Contém um link direcionando para o seu perfil no Instagram.
